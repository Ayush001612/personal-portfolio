Skills

HTML

HTML is the most basic building block of the web. It works alongside CSS and JavaScript to create websites. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript)

CSS

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media. CSS properties have different allowable values, depending on which property is being specified. For example, the color property can take various color values.

JavaScript

JavaScript (JS) is a high-level, interpreted programming language. It is one of the three core technologies of the World Wide Web, along with HTML and CSS. JavaScript is used to make web pages interactive and to provide dynamic content. It can also be used to create server-side applications.

Bootstrap

A free, open-source front-end framework for web development. It includes HTML, CSS, and optionally JavaScript-based design templates for typography, forms, buttons and navigation.

Tailwind

Tailwind CSS is a utility-first CSS framework that provides many classes for creating modern, responsive web designs. It's an open source framework that prioritizes the use of utility classes

React

React is a JavaScript library for building user interfaces. It is declarative, efficient, and flexible. React makes it easy to create interactive UIs by using a component-based approach.
